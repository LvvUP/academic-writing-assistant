# FAQ

## Is this a paper-writing tool?

It is an academic expression and structure assistant. It helps revise, translate, organize, and respond, but it should not replace the author's research work, evidence, or responsibility.

## Can it write related work without references?

It can draft a citation-free structure or use placeholders, but it must not invent references.

## Can it improve English academic style?

Yes. It can translate or polish academic English while preserving technical meaning and claim strength.

## Can it handle my field?

The current version includes field guidance for computer vision, AI, machine learning, medical imaging, remote sensing, NLP, robotics, data mining, bioinformatics, and materials science. Contributions for more fields are welcome.

## Does it require external services?

No. The Skill files and helper scripts are local. The scripts use only the Python standard library.
