# Detailed Roadmap

This document mirrors the root `ROADMAP.md` with more implementation detail.

## v0.1

- Build core Skill.
- Add modular references.
- Add field adapter for ten fields.
- Add terminology and structure checker scripts.
- Add README, examples, tests, and open-source files.

## v0.2

- Expand examples for abstract, introduction, related work, method, and experiment sections.
- Add more terminology variants.
- Add more tests for script edge cases.
- Improve bilingual output templates.

## v0.3

- Add cover letter and response letter templates.
- Add conference and journal style notes where they can be described safely.
- Add reviewer-response examples with missing-information placeholders.

## v0.4

- Add field packs for medical imaging, remote sensing, NLP, robotics, bioinformatics, and materials science.
- Improve section checker keyword rules for each field.
- Add manually curated evaluation prompts.

## v1.0

- Freeze stable Skill structure.
- Publish a contributor guide for field packs.
- Add richer manual evaluation documentation.
- Prepare release notes and plugin packaging guidance.
