# 中文到英文学术翻译示例

## Task

Chinese-to-English Academic Translation

## Field

Remote Sensing

## User Input

```text
请将以下中文段落翻译成适合 SCI 论文的英文表达，并给出术语表：

针对遥感图像中目标尺度变化大、背景复杂的问题，本文提出一种多尺度特征融合方法，以增强模型对不同尺度目标的表征能力。
```

## Expected Output

```markdown
## English Translation

To address large object-scale variations and complex backgrounds in remote sensing images, this study proposes a multi-scale feature fusion method to enhance the model's representation ability for objects at different scales.

## Terminology Choices

| 中文术语 | English Term | Note |
|---|---|---|
| 遥感图像 | remote sensing images | Standard term in remote sensing papers |
| 目标尺度变化 | object-scale variations | Keeps the object-level meaning |
| 多尺度特征融合 | multi-scale feature fusion | Common expression in vision-related work |
| 表征能力 | representation ability | Academic and concise |

## Translation Notes

- The translation keeps the problem-method-purpose logic.
- No dataset name, metric value, or performance claim was added.
```
