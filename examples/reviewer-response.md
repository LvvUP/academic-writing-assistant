# 审稿意见回复示例

## Task

Reviewer Response Drafting

## User Input

```text
请帮我回复审稿人意见，语气要礼貌、专业、不要过度辩解：

Reviewer: The novelty of the proposed method is unclear. The manuscript should better explain how it differs from existing feature fusion methods.
```

## Expected Output

```markdown
## Reviewer Comment Summary

The reviewer asks for a clearer explanation of the novelty of the proposed method, especially its difference from existing feature fusion methods.

## Response Draft

Thank you for this constructive comment. We agree that the novelty of the proposed method should be explained more explicitly. To address this concern, we propose revising the Introduction and Method sections to clarify the motivation and design differences of our feature fusion strategy. In particular, we will better explain how the proposed module integrates multi-scale information and how this design differs from conventional fusion operations.

## Manuscript Revision Statement

We will revise Section X, Page X, Lines X-X to clarify the motivation, design rationale, and differences from existing feature fusion methods.

## Suggested Revised Text

[Please insert a method-specific description here, such as the exact fusion path, attention design, or interaction mechanism. Avoid generic novelty claims.]
```
