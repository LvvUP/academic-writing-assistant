# 摘要写作示例

## Task

Abstract Writing

## Field

Computer Vision

## User Input

```text
请根据以下信息写一个英文摘要：

研究方向：小目标检测
问题：小目标特征弱，容易被复杂背景干扰
方法：加入上下文增强模块和多尺度特征融合模块
实验：目前还没有整理最终结果
```

## Expected Output

```markdown
## Draft

Small object detection remains challenging because small targets often contain limited visual cues and are easily affected by complex backgrounds. To address this issue, this work proposes a detection framework that integrates a context enhancement module with a multi-scale feature fusion module. The context enhancement module is designed to strengthen surrounding semantic information, while the multi-scale fusion module aims to improve feature representation across different object scales. Experiments will be conducted on [please provide datasets] using [please provide evaluation metrics]. The final quantitative results should be inserted after experimental validation.

## Writing Logic

- Starts from the general challenge of small object detection.
- Introduces the proposed modules without adding unsupported performance claims.
- Uses placeholders for missing datasets, metrics, and results.

## Missing Information

- Dataset names.
- Evaluation metrics.
- Main quantitative results.
- Comparison methods.
```
