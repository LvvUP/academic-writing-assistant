# Acceptance Checklist

The project is acceptable when all items are satisfied.

## Skill Structure

- [ ] `skills/academic-writing-assistant/SKILL.md` exists.
- [ ] `SKILL.md` has valid YAML frontmatter with `name` and `description`.
- [ ] Supporting reference files exist under `references/`.
- [ ] The Skill supports polishing, translation, expansion, merging, section writing, reviewer response, and terminology checking.

## Academic Integrity

- [ ] The Skill explicitly forbids fake citations.
- [ ] The Skill explicitly forbids fake experimental results.
- [ ] The Skill explicitly forbids fake datasets.
- [ ] The Skill tells the model to use placeholders when information is missing.

## Field Adaptation

- [ ] The Skill supports computer vision, AI, machine learning, medical imaging, remote sensing, NLP, robotics, data mining, bioinformatics, and materials science.
- [ ] The Skill can handle unspecified fields by using general academic style.

## README

- [ ] README explains what the project does.
- [ ] README includes quick examples.
- [ ] README includes installation or usage guidance.
- [ ] README includes academic integrity notes.
- [ ] README includes roadmap and contribution guidance.

## Scripts

- [ ] Helper scripts run without errors.
- [ ] Tests pass if scripts are included.
